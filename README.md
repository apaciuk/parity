# parity

Gem to split arrays by parity, odd/even

## Installation

Add lines to your application's Gemfile:

source 'https://gem.fury.io/xhostcom/' do
gem 'parity', '~> 0.0.3'
end

```ruby
gem "parity"
```

And then execute:

```bash
$ bundle
```
